Invisible Light Sources
This datapack adds invisible light blocks to the game, perfect for hidden lighting in your builds.

Features
Completely invisible light sources (light level 15)

Prevent hostile mob spawning

Don't ruin the aesthetics of your builds

Work in any dimension

No commands or cheats required

Survival-friendly

Usage
Place light blocks anywhere you need hidden lighting. They are invisible to the eye but fully function as light sources.

Requirements
Minecraft version: 26.2

Installation
Download the datapack archive

Place it in the datapacks folder of your world

Reload the world or run /reload command
